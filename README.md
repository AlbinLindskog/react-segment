# React hooks

A collection of useful React hooks. Contains:
- useStoredReducer
- useStoredState
- useAsync
- useDelayedAsync
- useDeepCompareMemo
- useDeepCompareEffect
- useOnClickOutside
- useScript


### Install
```bash
npm install --save git+git+https://github.com/AlbinLindskog/react-hooks.git
```

### Testing
Run all tests:
```bash
npm test
```
Run a specific tests:
```bash
npm test -- -t 'the tests description string'
```
